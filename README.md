Este projeto tem como objetivo reunir diferentes tipos de dados relacionados às doenças cardiovasculares para fins de estudo e análise em ciência de dados e inteligência artificial.

As doenças cardíacas estão entre as principais causas de morte no mundo, segundo a Organização Mundial da Saúde (OMS). Dessa forma, a análise de dados clínicos, textos científicos e imagens médicas pode contribuir para o desenvolvimento de ferramentas que auxiliem na prevenção, diagnóstico e tratamento dessas doenças.

O projeto está dividido em **três partes principais**, cada uma abordando um tipo diferente de dado utilizado em aplicações de análise de dados e aprendizado de máquina.

---

# 📊 Parte 1 — Dataset Estruturado de Saúde Cardiovascular

A primeira parte do projeto consiste na criação de um **dataset estruturado com informações clínicas simuladas** relacionadas à saúde cardíaca.

O dataset contém **mais de 100 registros**, representando pacientes com diferentes características clínicas.

## Variáveis presentes no dataset

- Idade
- Sexo
- Pressão arterial sistólica
- Pressão arterial diastólica
- Nível de colesterol
- Frequência cardíaca
- Histórico de doenças cardíacas
- Presença de sintomas (como dor no peito)

Essas variáveis são frequentemente utilizadas em estudos de **análise de risco cardiovascular** e **modelos de machine learning aplicados à saúde**.

## Objetivo

O objetivo desta etapa é simular um conjunto de dados clínicos que possa ser utilizado para:

- análise exploratória de dados (EDA)
- identificação de padrões em saúde cardiovascular
- desenvolvimento de modelos de classificação ou previsão de risco cardíaco

---

# 📚 Parte 2 — Coleta de Textos sobre Doenças Cardíacas

A segunda parte do projeto consiste na coleta de **textos científicos e informativos sobre doenças cardiovasculares**, sintomas, tratamentos e prevenção.

Foram reunidos **quatro textos em formato `.txt`**, que podem ser utilizados em atividades de **mineração de texto (NLP)**.

## Temas abordados nos textos

- Doenças cardiovasculares e saúde pública
- Principais sintomas de problemas cardíacos
- Tratamentos utilizados em cardiologia
- Prevenção e hábitos saudáveis para o coração

## Possíveis aplicações

Os textos coletados podem ser utilizados para:

- análise de frequência de palavras
- processamento de linguagem natural (NLP)
- criação de modelos de classificação de textos médicos
- extração de informações relevantes em saúde

## Fontes utilizadas

Os conteúdos foram baseados em informações provenientes de instituições e organizações de saúde reconhecidas, como:

- Organização Mundial da Saúde (WHO)
- Ministério da Saúde do Brasil
- Biblioteca Virtual em Saúde (BVS)
- Sociedade Brasileira de Cardiologia
- Organização Pan-Americana da Saúde (OPAS)

---

# 🩻 Parte 3 — Dataset de Imagens Médicas

A terceira parte do projeto consiste na coleta de **imagens médicas relacionadas a exames cardiológicos**.

Foram reunidas **mais de 100 imagens** em formato `.jpg` ou `.png`.

## Tipo de exame escolhido

As imagens coletadas correspondem a exames de:

- ECG (Eletrocardiograma)

O eletrocardiograma é um exame amplamente utilizado para avaliar a atividade elétrica do coração e auxiliar no diagnóstico de diversas condições cardíacas.

## Possíveis aplicações

O conjunto de imagens pode ser utilizado para:

- classificação de imagens médicas
- treinamento de modelos de visão computacional
- estudos de reconhecimento de padrões em exames cardíacos
- desenvolvimento de sistemas de apoio ao diagnóstico
